MAO NOTE #02

「新年のご挨拶とチーム紹介」

MAO NOTE 1月号

皆さんこんにちは、MineNetworkアドミンのナズナです！
そして... あけましておめでとうございます！！
2021年も、いろんなコンテンツをご用意しておりますので、ぜひお楽しみに！

MAO NOTE とは？

さて、このMAO NOTEを見てくれて、ありがとうございます！
ここでは、MAOを遊んでくれている皆さんにもっと私たちのコンテンツを知ってもらうため、
どんな思いや狙いがあって開発をしているのかをもっと知ってもらうために、
普段は語られない建築に込められた思いや、プレイヤーを恐怖に陥れるあのMOBのバックボーンなど、
MAOにまつわる沢山の面白い話や泣ける話(?)を定期的に、皆さんにブログのような形でお届けします！
ライターは開発を行っているスタッフが毎回変わって担当しますので、お楽しみに！

さて、記念すべき2021年初回の今回は、MAOスタッフの役職をご紹介します！

ビルダー / Builder

まずは、MAOの景観を担当する方々、ビルダーです！
ビルダーは、MAOの建物や、山や海、川などの地形を作る職人さんです。
MAOでは、一層ごとに、もっと言えば１つの街ごとにコンセプトや設定があります。
そういったコンセプトをもとに多種多様な景観を作り出すことはまさに、彼らの卓越した技といえると思います。
(まあ、自分は建築すると必ず豆腐になりますが...)

クリエイター / Creator

続いて、クリエイターの方々です！
彼らはMAOがゲームとして成立するように必要な多くのコンテンツを担っています。
大まかに、MOBやアイテム、クエストやソードスキルなど、細かく言えばもっとあります。
実際のところ、自分もクリエイター班にはいつも驚愕してます。
何がすごいって、彼らアイデアがあふれ出て止まらないんですよ(笑)
いつもクリエイター班の会議に行くと層の設定やら、剣についての設定をすっごいまじめに、でも楽しそうに話してるんですよね。
いやー、ほんと彼らには脱帽です...!

デザイナー / Designer

さてさて次は～、デザイナー班！
彼らも負けず劣らずすごいですよ...!
OGTに参加してくれた方ならわかると思います、プロモーターソード！
さらには、ソードスキルのアイコン。
デザイナー班はテクスチャから始まってアイコン、さらには3Dモデルまで。
いろいろなデザインを統括しています。
自分は、イラストとか絵は描けない人なのでまじで尊敬してます...

デベロッパー / Developer

そして、MAOを基礎から支える、デベロッパー！
MAOの象徴でもあるソードスキルを筆頭に、クエストや鍛冶システムなど。
皆さんがもっと楽しく遊べるようにMAOにシステムを追加する根幹的存在です。
この記事を書いている今もコードを書いているらしく、報告がたくさん届いてます(笑)
今後も、もっともっと楽しめるシステムを作っていきますのでよろしくお願いします！

モデレーター / Moderator

また、開発をしているスタッフとは別にモデレーターと呼ばれる方々もいます！
モデレーターはユーザーの皆さんがサーバーで楽しく遊んでいただいけるように、
皆さんのサポートをしたり、適切な処罰をするチームです！
普段はあまり目立ちませんが、皆さんが楽しく遊べるために尽力しています！

2021年もMAOをよろしくお願いします！

いかがでしたでしょうか、この他にも運営全体の統括をするアドミンもいて、全6つの役職でMAOの開発は進んでいます。
MAOでは、ユーザーとの交流を一層大切にしようという開発指針を取っています。
サーバー内や、Discordで我々スタッフを見かけたら、ぜひ積極的に交流をしてみてくださいね！
2021年も、MineNetwork、MineArtOnline をよろしくお願いします！

さて、今回のプロジェクトメッセージは、このあたりで。次回の更新は大体一か月後です！お楽しみに！
ライターは、MineNetworkアドミンのナズナでした。ありがとうございました！
